It's easy to create this type of banner- just change out the URL and alt info in the template. To change the image from a banner that spans the entire screen to one that is the width of the main content, use the class `.container` instead of `.container-fluid`. For other images within the content, you can use markdown like so: ``` ![Strongbad Octocat](https://octodex.github.com/images/strongbadtocat.png "Don't give me none of that crosstalk!") ``` This example shows up like so: ![Stongbad Octocat](https://octodex.github.com/images/strongbadtocat.png "Don't give me none of that crosstalk!") To make images smaller sizes, add the following to the title: `width=200;` ``` ![Strongbad Octocat](https://octodex.github.com/images/strongbadtocat.png "Don't give me none of that crosstalk! width=200;") ``` The resized image will look like this: ![Stongbad Octocat](https://octodex.github.com/images/strongbadtocat.png "Don't give me none of that crosstalk! width=200;") To align the images, add the following to the title: `align=center;` ``` ![Strongbad Octocat](https://octodex.github.com/images/strongbadtocat.png "Don't give me none of that crosstalk! width=200; align=center;") ``` The aligned image will look like this: ![Stongbad Octocat](https://octodex.github.com/images/strongbadtocat.png "Don't give me none of that crosstalk! width=200; align=center;") _All images that are left or right aligned will have a float and the text will wrap around the image_ All of these types of images will be displayed within the width of the main content.
